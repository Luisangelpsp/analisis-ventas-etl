# Proyecto ETL – Análisis de Ventas

Este proyecto implementa un pipeline ETL para el análisis de ventas utilizando Python y MySQL.

## Objetivo
Extraer datos desde archivos CSV, transformarlos mediante limpieza y normalización, y cargarlos en una base de datos relacional para su análisis.

## Contenido del repositorio
- etl_analisis_ventas.py: script ETL en Python
- crear_bd.sql: script de creación de la base de datos
- Archivos CSV: fuentes de datos de entrada
- Evidencias: capturas de consultas y resultados
- Diagrama ER del modelo de datos

## Tecnologías utilizadas
- Python
- Pandas
- SQLAlchemy
- MySQL

## Flujo ETL
1. Extracción de datos desde archivos CSV
2. Limpieza de duplicados y valores nulos
3. Normalización de formatos
4. Cálculo del total de ventas
5. Carga de datos en MySQL
